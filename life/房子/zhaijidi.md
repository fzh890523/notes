
# 关于宅基地以及宅基地房屋买卖

```
土地性质是城镇国有土地还是农村集体土地。
如果是农村集体土地的话，那么应该是宅基地性质。
宅基地建的房子目前属于小产权房，也就是房子归自己，但土地使用权不能自由转让（只能在本集体经济成员之间，比如本村）。 也就是说卖给外地人的话是不合法的。可行的方式是购买者将户口转入本集体经济组织（如本村）然后再进行房屋买卖。
此外，除了继承以外，其他方式获得的宅基地不得超过一户一宅。
```
