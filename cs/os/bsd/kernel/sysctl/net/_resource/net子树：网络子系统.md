

# net子树：网络子系统

net子树主要用于调整 FreeBSD 网络子系统的行为，它包括下列子树：

- [net.bpf](https://wiki.freebsdchina.org/doc/s/sysctl/net.bpf)
- [net.inet](https://wiki.freebsdchina.org/doc/s/sysctl/net.inet)
- [net.inet6](https://wiki.freebsdchina.org/doc/s/sysctl/net.inet6)
- [net.isr](https://wiki.freebsdchina.org/doc/s/sysctl/net.isr)
- [net.link](https://wiki.freebsdchina.org/doc/s/sysctl/net.link)
- [net.local](https://wiki.freebsdchina.org/doc/s/sysctl/net.local)
- [net.route](https://wiki.freebsdchina.org/doc/s/sysctl/net.route)
- [net.wlan](https://wiki.freebsdchina.org/doc/s/sysctl/net.wlan)
