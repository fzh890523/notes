

# tutorial



## 参数



* 没有找到指定*输出的字段*的参数

* 非交互模式（批量模式） `-b`

* 线程模式（显示线程而不是进程） `-H`

  恶心的是，只打线程id了，看不出线程和进程的关系

* 刷新的时间间隔 `-d ${secs}`

* 指定排序列 `-o ${field}`

  支持顺序倒序（负号 `-`）

  如`-o -%CPU`

* 显示完整命令（含参数）： `-c`

  默认只显示 命令本身



## 场景



### 查看当前cpu消耗



```shell
top -d 1 -H -b -n 1 -c -o -%CPU
```
