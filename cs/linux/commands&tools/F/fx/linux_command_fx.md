# 使用

## 一些关键操作

### 复制

ref: https://github.com/antonmedv/fx/blob/master/doc/doc.md#selecting-text

交互模式下键盘方向键和鼠标会带来特殊操作如展开、收起，所以无法直接做到 **对当前可见内容进行复制** ，需要的操作如下：

| Key              | Terminal      |
| ---------------- | ------------- |
| `Option`+`Mouse` | iTerm2, Hyper |
| `Fn`+`Mouse`     | Terminal.app  |
| `Shift`+`Mouse`  | Linux         |
