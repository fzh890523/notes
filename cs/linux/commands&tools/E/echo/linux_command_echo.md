

## 重要的参数选项

* `-e` 解释`\t`这样的（为）转义字符

  ```sh
  v="123\n456"
  echo "${#v}"  # 8
  # 说明shell本身并没做转义，而是需要类似如下：
  v1="123
  456"
  echo "${#v1}"  # 7
  
  # or
  v1=$(echo -e "123\n456")
  ```

  

* `-E` 不解释`\t`这样的转移字符。 默认行为
  这个默认行为可能被调整、配置修改

* `-n` 不输出末尾的换行符。 默认会添加换行符

  > 有的shell会强制换行，导致看不出这个差别来
